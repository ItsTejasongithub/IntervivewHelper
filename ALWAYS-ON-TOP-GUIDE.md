### 2. **Always-On-Top Solution**

#### Option A: Chrome Flag (Basic - Already Working)
The app uses `--always-on-top` Chrome flag. It works but has limitations when clicking other windows.

#### Option B: AutoHotkey (Recommended - Best Solution!)
Install AutoHotkey for **persistent always-on-top** that works even when clicking Zoom/Teams.

**Installation Steps:**
1. Download AutoHotkey: https://www.autohotkey.com/download/
2. Install AutoHotkey v2 or v1.1
3. Double-click `launch-app.bat` - AutoHotkey will start automatically!

**AutoHotkey Features:**
- ‚úÖ Window stays on top even when clicking Zoom/Teams/Meet
- ‚úÖ Auto-detects the AI Assistant window
- ‚úÖ Hotkey: `Ctrl+Alt+T` to toggle always-on-top for any window
- ‚úÖ Hotkey: `Ctrl+Alt+Q` to exit the helper script

## üîß Troubleshooting:

### Window goes behind when clicking?
- **Solution 1**: Install AutoHotkey (see above)
- **Solution 2**: Press `Ctrl+Alt+T` on the app window to force always-on-top
- **Solution 3**: Use a second monitor for the assistant

### Can't see the window?
- It might be too transparent! Open Settings and increase opacity to 100%

### Window disappeared?
- Check if it's off-screen - drag it back or restart the app
